# pi-opencode-theme

[![npm version](https://img.shields.io/npm/v/pi-opencode-theme)](https://www.npmjs.com/package/pi-opencode-theme)
[![npm downloads per week](https://img.shields.io/npm/dw/pi-opencode-theme?label=downloads%2Fweek)](https://www.npmjs.com/package/pi-opencode-theme)

An opencode-inspired theme for [pi](https://pi.dev).



## Install

```bash
pi install npm:pi-opencode-theme
```

Or from GitHub:

```bash
pi install git:github.com/cruzluna/pi-opencode-theme
```

## Use

Select `opencode` in `/settings`, or add this to your pi settings:

```json
{
  "theme": "opencode"
}
```

## Local development

```bash
pi -e /absolute/path/to/pi-opencode-theme
```

Then select `opencode` in `/settings`.
